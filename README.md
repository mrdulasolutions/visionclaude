**ClaudeVision** turns your iPhone (or Meta Ray-Ban glasses) into Claude's eyes and ears. Point your camera, speak naturally, and Claude sees what you see ‚ÄĒ then responds with voice, using your connected tools like email, calendar, and more.

## How It Works

```
Phone Camera  ‚Üí  Gateway Server  ‚Üí  Claude API
Your Voice    ‚Üí    (your Mac)    ‚Üí  MCP Tools
Claude Reply  ‚Üź                  ‚Üź  (email, etc.)
```

1. **You speak** ‚ÄĒ on-device speech recognition transcribes your voice
2. **Camera captures** ‚ÄĒ latest frame grabbed as JPEG (~1fps)
3. **Gateway sends** ‚ÄĒ text + image to Claude via the Anthropic API
4. **Claude responds** ‚ÄĒ with vision understanding + tool actions
5. **You hear** ‚ÄĒ text-to-speech reads Claude's response aloud
6. **Loop repeats** ‚ÄĒ continuous conversation, hands-free

## Architecture

### Gateway Server (Node.js)

Runs on your Mac. Bridges the iOS app to Claude's API and routes tool calls through MCP servers.

```
server/
‚Ēú‚ĒÄ‚ĒÄ src/
‚Ēā   ‚Ēú‚ĒÄ‚ĒÄ index.ts              # Express entry point (port 18790)
‚Ēā   ‚Ēú‚ĒÄ‚ĒÄ claude-client.ts      # Claude Messages API + vision + tool loop
‚Ēā   ‚Ēú‚ĒÄ‚ĒÄ mcp-manager.ts        # MCP server lifecycle & tool discovery
‚Ēā   ‚Ēú‚ĒÄ‚ĒÄ conversation.ts       # In-memory conversation store
‚Ēā   ‚ĒĒ‚ĒÄ‚ĒÄ routes/
‚Ēā       ‚Ēú‚ĒÄ‚ĒÄ chat.ts           # POST /chat ‚ÄĒ main endpoint
‚Ēā       ‚Ēú‚ĒÄ‚ĒÄ health.ts         # GET /health ‚ÄĒ status check
‚Ēā       ‚Ēú‚ĒÄ‚ĒÄ tools.ts          # GET /tools ‚ÄĒ list MCP tools
‚Ēā       ‚ĒĒ‚ĒÄ‚ĒÄ config.ts         # GET/POST /config ‚ÄĒ runtime config
```

**Key feature:** Auto-discovers MCP tools from your Claude Desktop config. Any MCP server you add to Claude Desktop automatically becomes available through the gateway.

## API Endpoints

| Endpoint | Method | Description |
|---|---|---|
| `/health` | GET | Server status + connected MCP tool count |
| `/tools` | GET | List all discovered MCP tools |
| `/chat` | POST | Send text + images, get Claude's response |
| `/config` | GET/POST | View or update system prompt, model |

### Chat Request

```json
{
  "text": "What do you see?",
  "images": ["<base64 JPEG>"],
  "conversation_id": "optional-session-id"
}
```

### Chat Response

```json
{
  "text": "I can see a laptop on a desk with...",
  "tool_calls": [{"name": "read_emails", "result": {...}}],
  "conversation_id": "generated-session-id"
}
```

## MCP Tools

ClaudeVision automatically discovers tools from your Claude Desktop configuration at:

```
~/Library/Application Support/Claude/claude_desktop_config.json
```

Any MCP server configured there will be available through voice commands. For example:
- "Check my email" ‚Üí uses email MCP tools
- "What's on my calendar?" ‚Üí uses calendar MCP tools
- "Send a message to..." ‚Üí uses messaging MCP tools
